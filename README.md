# AutoClaving-Calculator-Growtopia-Game
Ini adalah sebuah project untuk membantu para pemain growtopia untuk menghitung Autoclave yang di hasilkan untuk menentukan berapa Stit yang akan di dapatkan dan juga mendapatkan gambaran berapa item dan modal untuk memulai profitan autoclave ini

## How To RUN

Sistem ini hanya pengembangan nya saja dan akan saya terus update untuk kedepan nya

### Fitur
* Autoclave Calculator menghitung Sampah surgical item menjadi sebuah Stit
* Menambahkan fitur inventory selayaknya sedang ada di dalam game Growtopia

**Update kedepan nya saya akan mengembangkan sebuah website agar orang awam bisa mengakses nya**
